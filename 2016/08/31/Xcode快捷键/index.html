Xcode应用程序的快捷方式

By MrXiaoYang

Published 2016-08-31

Contents
  1. 1. Xcode相关：
    1. 1.1. 打开Xcode偏好设置（Preferences） cmd ＋ 逗号（,）
    2. 1.2. 隐藏Xcode cmd ＋ H
    3. 1.3. 隐藏其他文件（非当前窗口显示的文件为其他文件，非鼠标光标所指文件） cmd ＋ Alt ＋ H
    4. 1.4. 退出Xcode cmd ＋ Q
  2. 2. 文件相关：
    1. 2.1. 添加一个新桌面(New Tab) cmd + T
    2. 2.2. 添加一个新窗口((New Window)当前项目拷贝一份显示) Shift + cmd + T
    3. 2.3. 创建一个新文件(New File) cmd + N
    4. 2.4. 创建一个新项目(New Project) Shift + cmd + N
    5. 2.5. 创建一个新工程(New Workspace) Ctrl + cmd + N
    6. 2.6. 创建一个虚拟文件夹(New Group) Ctrl + Alt + N
    7. 2.7. 快速打开一个选中的文件夹 cmd + O
    8. 2.8. 快速查找项目中的.h或.m文件 Shift + cmd + O
    9. 2.9. 关闭当前窗口 cmd + W
    10. 2.10. 关闭所有窗口 cmd + Alt + W
    11. 2.11. 关闭当前桌面(Tab) Shift + cmd + W
    12. 2.12. 关闭除当前桌面以外的其它桌面(Tab) Shift + cmd + Alt + W
    13. 2.13. Close Document ⌃⌘W
    14. 2.14. Save ⌘S
    15. 2.15. Save All ⌥⌘S
    16. 2.16. Save Multiple ⌥⇧⌘S
    17. 2.17. Save As ⇧⌘S
    18. 2.18. Commit… ⌥⌘C
    19. 2.19. Update…. ⌥⌘X
    20. 2.20. Update All ⌃⌥⌘X
    21. 2.21. Create Snapshot ⌃⌘S
    22. 2.22. Page Setup ⇧⌘P
    23. 2.23. Print ⌘P
  3. 3. 编辑相关(Edit)
    1. 3.1. Undo ⌘Z
    2. 3.2. Redo ⇧⌘Z
    3. 3.3. Cut ⌘X

    14. 3.14. Find and Replace… ⌥⌘F

    16. 3.16. Find Previous ⇧⌘G

    18. 3.18. Use Selection for Replace ⇧⌘E
    19. 3.19. Filter in Navigator ⌥⌘J
